## 🚀 About Me

- 🏥 Currently shipping **HIPAA-compliant LLM agents** at **Kaiser Permanente** — RAG over 2M+ clinical documents, multi-step agent workflows on Vertex AI, and FastAPI services sustaining **500 QPS** for member-facing features.
- 🤖 Deep work in: **RAG pipelines, LangGraph agents, LLM fine-tuning (LoRA / PEFT), multi-modal models, GPU inference (vLLM, Triton, TensorRT)**.
